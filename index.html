Hall of Eternity
Tower Defense, Collectible Card Game
Java 2D Game Framework
Previous Next
Hall of Eternity is a tower defense game where you protect the evil summoner from various 'hero' classes. This is done by building a deck of various monster, curse, and trap cards. Monster cards will summon units you are able to place on a grid to fight adventurers, curse cards have instant effects on the board, and trap cards create permanant traps that affect 'hero' units.
