Hall of Eternity is a tower defense game where you protect the evil summoner from various 'hero' classes. This is done by building a deck of various monster, curse, and trap cards. Monster cards will summon units you are able to place on a grid to fight adventurers, curse cards have instant effects on the board, and trap cards create permanant traps that affect 'hero' units.

Redemption is a 3D Roguelike from a third person point of view. The game has 3 randomly generated levels, each based on one of the seven deadly sins with thematic enemies and bosses. There are 4 classes to choose from with distinct skills and a class identity. There are also 4 weapon types for different combat styles and over 10 different modifiers for rare weapons found in the dungeon.

The class project for this game has recently finished. Only the first 3 screenshots are recent. I will add some GIFs when I get back from vacation! :) (12/21/2017)

This is a more recent project that I've started with an artist and sound designer. The core gameplay is a match 3 puzzler in which you must match monster tiles to keep them from hurting your party. I've fleshed out a reasonably expansive job and party system that I'd like to implement further along in development. Different classes would be able to spawn various matching tiles or affect the matches of other tiles in some way. I've been working on this a bit each week and invite you to check out my repo for changes, I'll also be updating this site semi-regularly during these early stages.

For now I have also included images of much smaller Löve projects including a basic TicTacToe interface and some early map algorithm design for Redemption.
